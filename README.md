## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/abchatra/makecode-zoo-demo1** and import

## Edit this project ![Build status badge](https://github.com/abchatra/makecode-zoo-demo1/workflows/MakeCode/badge.svg)

To edit this repository in MakeCode.

* open [https://minecraft.makecode.com/](https://minecraft.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/abchatra/makecode-zoo-demo1** and click import
